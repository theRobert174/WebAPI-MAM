# Medical Appointment Manager

## Database creation and update (Packet Manager Console)
```
Update-Database
```

## Create new file to alter the Database
```
Add-Migration
```

## Remove last migration
```
Remove-Migration
```

##Tech Stack

**Server:** C#, EntityFrameworkCore, SwaggerUI
